# Superstore-sales-analytics
The purpose of this project is to provide a comprehensive process of sales data analysis from data pre-processing using the Python programming language (Pandas library) up to the development of an interactive dashboard in Power BI for business intelligence purposes. 
 Sales Analysis Dashboard

 Overview
This project presents an end-to-end sales data analysis workflow using Python (Pandas) and Power BI. The goal is to analyze business performance, identify profit leaks, and provide actionable insights for better decision-making.

 Problem Statement
Businesses often focus on revenue without analyzing profitability. This project aims to uncover hidden losses, optimize discount strategies, and identify high-performing areas.

 Tools & Technologies
- Python (Pandas)
- Power BI

 Data Cleaning (Python)
- Handled missing values
- Removed duplicate records
- Standardized data formats
- Prepared clean dataset for analysis



 Dashboard Features
- Sales trends over time
- Regional performance analysis
- Product category insights
- Customer segmentation
- Discount vs Profit analysis


 Key Insights
- West region has highest sales, but Central has lower profit margins  
- Tables category causes major losses  
- High discounts negatively impact profitability  
- Technology products (Copiers) generate highest profit  

 Recommendations
- Limit discounts to improve margins  
- Focus on high-profit products  
- Improve strategy in loss-making regions  

Project Workflow
Raw Data → Data Cleaning → Analysis → Dashboard → Insights → Recommendations
